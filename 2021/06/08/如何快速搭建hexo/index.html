手把手教你从0开始搭建自己的个人博客

第一步下载node.js和git

node.js官网

git的官网

因为win都是无脑安装的所以你自己点下去就没问题

在cmd下出现输出node -v 出现版本就安装成功了

git 安装需要你们进一步学习和了解本人就不深入了因为git安装本身就可以写一篇博客了以后会补上。推荐了解git后来玩hexo

第二部安装淘宝的cnpm 管理器

在cmd下安装，安装成功后输出cnpm -v测试安装，基本不是断网都可以安装好

npm install -g cnpm--registry=http://registry.npm.taobao.org

安装hexo框架（都是在cmd下完成）

cnpm install -g hexo-cli

本人已经在win下安装过hexo所以我就直接查看

不用担心会出现安装失败的情况只有你安装或者断网的情况下

本地安装hexo

随便在你的盘上创建一个blog的目录，进入目录

进入git bash here 输出hexo init。如果出现以下报错。

在cmd或者就在git下输入npm rebuild node-sass 或者cnpm rebuild node-sass之后安装就可以了

如果还是有错误的话我直接推荐你去开个梯子(vpn)直接安装报错不用管的他会直接下载完用就完事了

输出hexo s 就可以了

进入localhost:4000就部署成功了
